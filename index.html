Mac 上用 Hexo / Github Pages 搭建个人 Blog

发表于 2017-04-21 | 分类于 CS

第一篇 Blog 主要纪录下这次的搭建问题。

  1. 安装 Hexo。如建立不成功，使用「sudo」执行
    sudo npm install -g hexo-cli

  2. 「 localhost:4000 」error:
    在 _config.yml 文件里，把 server 的 port 值随便换一下。

  3. SSH Key 部分：
    建立在默认文件夹下，不用输入，直接回车。

  4. 找不到 git！
    这个 error 花了些时间，后来发现建立 git deploy 的正确命令应是

    npm install hexo-deployer-git --save

    而之前一直把「deployer」输入成了「developer」，看教程没带脑子的错。

  5. Repositories:
    搭建好后一直是 404 page not found，最后发现是 repositories 的名字问题，看了n次这个 solution 却一直忽略，最后突然醒悟。
    正确命名应该符合： username/username.github.io
    我的后缀没问题，主要是第二个 username 处换了别的命名，在 github repositories 里的齿轮图标 Settings 里 rename 就可。然后在 _config.yml 里改一下 deploy 里的 repository。

  6. Tags 页面 404 问题：
    需要自己生成 .md 文件。使用 hexo new page "tags" 命令，source 文件夹里生成 tags/index.md。

  7. 编辑文章 语法问题：
    很多网站都有可用语法生成，包括一些图站都有自动转换 markdown 语法的选项。
    目前选用图站：SM.MS
    语法参考网站：小书匠
    这两个都很好上手。

网上教程很多，基本流程此处不赘述了。

技术之外

想写 Blog 有段时间了，检索一圈后，决定自己搭建，初步体验感很不错。激发兴趣的同时，对后面继续做 AI 也有助益。
